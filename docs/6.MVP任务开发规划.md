##### 任务 1.2: 超级画布 (Canvas) 开发
**开发内容**:
- 基于 RN Skia 实现触控画板
- 集成 `perfect-freehand` 算法，将触控点转为平滑 SVG Path
- 实现防手掌误触 (Palm Rejection) 启发式算法
- 实现工具栏: 5色选择、3种笔刷粗细、橡皮擦、撤销、清空
- 画布内容导出为 PNG Base64

**开发步骤**:
1. 创建 `CanvasScreen` 组件，初始化 Skia Canvas
2. 监听 `onTouchStart/Move/End` 事件，收集触控点
1. 实现 Palm Rejection:
   - 判定1: `touch.radiusX * touch.radiusY > PALM_THRESHOLD` → 忽略
   - 判定2: 同时触控点 > 3 → 暂停绘制
4. 将触控点传入 `perfect-freehand` 生成平滑路径
5. 使用 Skia `Path` 绘制，支持不同颜色和粗细
6. 实现撤销栈 (undo stack)，最多保留 20 步
7. 导出画布为 PNG: `canvas.makeImageSnapshot().encodeToBase64()`

**验收清单**:
- [ ] 在 iPad 上画线无明显延迟 (< 16ms per frame)
- [ ] 手腕靠在屏幕上画画时不产生乱线
- [ ] 5 种颜色切换正常，笔触有平滑的粗细变化
- [ ] 撤销/清空功能正常
- [ ] 导出的 PNG 图片清晰完整

#### Sprint 2 (Week 3-4): AI 渲染管线核心开发

##### 任务 2.1: 多模态意图解析与智能路由
**开发内容**:
- FastAPI 接口: 接收图片 + 音频，调用 ASR 转文字

**开发步骤**:
1. 实现 `POST /stories/{id}/scenes` 接口，接收 multipart 上传
2. 上传原始文件至 OSS `original/` 目录
1. 调用 ASR API (Whisper / 火山引擎) 将音频转文字
4. 将 `(原图, 转写文字)` 送入 GPT-4o，执行语义纠错
5. 实现路由评分 Prompt，返回 `complexity_score` + `render_route`
6. 实现 AI 导演 Prompt 模板，生成英文运镜指令
7. 结果写入 `scenes` 表，通过 WebSocket 推送 `narration_ready`

**验收清单**:
- [ ] 上传图片+音频后 3 秒内返回修正旁白
- [ ] 故意说"粗苹果"，画布有苹果形状时，能纠正为"吃苹果"
- [ ] 静态场景(如"这是一座山") score < 4，路由到 parallax_2d
- [ ] 动作场景(如"恐龙在跑") score > 6，路由到 i2v_api
