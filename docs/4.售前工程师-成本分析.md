### 一、 竞品 API 现状剖析（2026年语境）

目前市面上的顶尖大模型 API（如可灵、海螺、位元组）确实非常强大，但它们是**「通用型基础设施」**，并非为「儿童 UGC 场景」设计的。

*   **位元组跳动 (PixelDance / Seaweed)**：动态一致性极佳，适合做复杂的运镜，但对 Prompt 的精准度要求极高。
*   **可灵 (Kling)**：物理规律模拟最强，画质极高，但 API 呼叫的排队时间通常较长，且生成价格偏贵。
*   **海螺 (Minimax)**：对二次元、动画风格的理解极佳，生成速度快，但对抽象涂鸦（ControlNet 边缘控制）的支援度较弱，容易出现「AI 幻觉（如多出肢体）」。

**如果单纯套用它们的 API（The "Lazy" Wrapper Approach）：**
*   **优点**：开发极快（只需工程师看几天 API 文件就能接上）；不需要自己维护 GPU 丛集；能蹭到大厂模型升级的红利。
*   **致命缺点**：
    1.  **听不懂童言童语**：大模型需要「工业级 Prompt」。小孩说「打恐龙」，API 会真的生成一只被打的写实恐龙，而不是可爱的绘本风。
    2.  **无法锁定角色 IP**：第一页画的狗和第二页生成的狗长得完全不一样，故事直接断层。
    3.  **盲盒效应（抽卡）**：大约有 20%-30% 的机率会生成肢体扭曲的「恐怖谷」画面，这对儿童产品是毁灭性的打击。

---

影片生成的底层依赖大量的 GPU（H100/B200）推论算力，因此价格远高于纯文字或图片。以下是目前业界最顶尖的 API 单次呼叫（生成一段约 **5秒、720p/1080p** 影片）的平均企业级定价：

*   **快手 可灵 (Kling API)**：画质与物理规律最强。
    *   标准版 (Standard)：约 **$0.15 - $0.20 USD** / 次。
    *   专业版 (Pro, 1080p)：约 **$0.40 - $0.50 USD** / 次。
*   **稀宇 海螺 (Minimax API)**：动漫风格佳，生成速度快。
    *   平均定价：约 **$0.15 USD** / 次。
*   **字节跳动 (Seaweed / PixelDance)**：运镜强，多视角一致性好。
    *   平均定价：约 **$0.12 - $0.18 USD** / 次。
*   *(作为对比)* **OpenAI Sora (若开放)**：预估极高，可能在 $1.00 USD / 次以上。

**基准线确立**：为了方便计算，我们取目前市场最具竞争力的平均中位数：**单次呼叫 I2V API 的硬成本为 $0.15 USD**。

### 二、 残酷对比：生成一部「3 页微电影」的真实成本

我们设定一个标准产品交付单位：**一部影片 = 包含 3 个场景（页），每个场景约 4-5 秒，总长 15 秒，含配音与配乐。**

#### 方案 A：单纯 API 套壳方案 (The "Lazy Wrapper" Approach)
*逻辑：前端收到孩子的 3 张图和语音，直接无脑传给 Kling / Minimax API 产出 3 段影片再拼接。*

1.  **基础呼叫成本**：3 个场景 × $0.15 = $0.45 USD。
2.  **隐藏的「抽卡税」(The Reroll Tax - 致命伤)**：
    小孩子画的图和说的话非常发散。未经工程化约束的大模型，生成肢体变形、多出腿、或者物体融化的机率极高。为了保证交付给用户的影片能看，系统或用户平均需要**重试（抽卡） 2 次**才能得到 1 个勉强可用的场景。
    *实际呼叫次数*：3 个场景 × 2 次抽卡 = 6 次 API 呼叫。
    *抽卡后成本*：6 × $0.15 = **$0.90 USD**。
3.  **其他周边成本**：ASR (语音辨识) + LLM (文字润饰) + TTS (配音) ≈ **$0.02 USD**。

🔴 **方案 A 总成本：$0.92 USD / 部影片** 

---

#### 方案 B：我们的「魔法渲染引擎」(The Engineered Pipeline)
*逻辑：透过智能路由，把好钢用在刀刃上；透过 VLM 拦截，彻底消灭无意义的抽卡浪费。*

1.  **智能路由分流 (Smart Routing)**：
    我们的系统不会把 3 页全部丢给昂贵的 I2V API。根据 LLM 语义解析，我们做算力分流：
    *   **第 1 页（风景/静态介绍）**：走自建的 2.5D 深度视差管线。伺服器算力成本：**$0.005 USD**。
    *   **第 2 页（简单挥手/跳跃）**：走自建 2D 骨骼动画 (Animated Drawings)。伺服器算力成本：**$0.005 USD**。
    *   **第 3 页（复杂物理互动，如恐龙喷火）**：呼叫顶级 I2V API (如 Minimax)。成本：**$0.15 USD**。
    *基础生成成本*：$0.005 + $0.005 + $0.15 = **$0.16 USD**。
2.  **重塑与角色一致性 (ComfyUI / Serverless GPU)**：
    在送入影片生成前，我们使用自建的 Flux + ControlNet 节点将小孩的草图重塑为 3D 图片。使用 RunPod 等 Serverless 服务，单张图推论成本约 $0.015。3 张图 = **$0.045 USD**。
3.  **VLM 视觉巡检兜底 (Auto-QA)**：
    使用 GPT-4o-mini 或 Claude 3 Haiku 检查第 3 页的影片是否破图。检查成本：**$0.005 USD**。
    *消灭抽卡税*：因为我们已经在生图阶段锁定了骨架（ControlNet），且影片 Agent 下达了精确的运镜 Prompt，I2V 的失败率极低。即使失败，我们**直接降级回免费的 2.5D 方案**，不再花 $0.15 重抽。因此抽卡成本 = **$0.00 USD**。
4.  **其他周边成本**：ASR + LLM + TTS + FFmpeg 拼接 ≈ **$0.02 USD**。

🟢 **方案 B 总成本：$0.235 USD / 部影片**
